# UrbanStore

UrbanStore es una aplicación de Android para gestionar un carrito de compras y realizar pedidos.

## Características

- Añadir productos al carrito
- Ver historial de pedidos
- Eliminar pedidos
- Finalizar compras

## Requisitos

- Android Studio
- Firebase para autenticación y base de datos
